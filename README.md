## 📖 Overview

一个涵盖多领域的 **Python 单脚本项目集合**，包含人工智能、机器学习、数据分析、仿真模拟、智能系统等多个领域的实用代码示例。

本仓库旨在提供：
- 🧠 丰富的 AI 与机器学习实践案例
- 🔬 科学计算与数值方法实现
- 🌌 创意性的仿真模拟系统
- 🏠 实用的智能系统原型

## ✨ 核心特点

- **即开即用** - 每个脚本都是独立完整的，直接运行即可
- **多领域覆盖** - 从人工智能到工业工程，涵盖9大领域
- **代码质量** - 遵循 PEP 8 规范，代码结构清晰
- **持续更新** - 不断添加新的脚本和功能

## 🚀 Quick Start

### 环境要求

```bash
Python 3.7+
pip package manager
```

### 安装依赖

```bash
# 克隆仓库
git clone https://github.com/jiarobot/python_demo.git
cd python_demo

# 安装核心依赖
pip install numpy scipy pandas matplotlib seaborn

# 安装机器学习库
pip install scikit-learn tensorflow torch

# 安装GUI框架
pip install PyQt5 PyQtChart

# 安装其他工具
pip install requests beautifulsoup4 joblib
```

### 运行脚本

```bash
# 直接运行任意脚本
python 神经元.py
python 智能咖啡.py
python 宇宙变迁.py

# 部分脚本支持命令行参数
python 智能咖啡.py --help
```

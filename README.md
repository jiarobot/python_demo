## 📖 Overview

一个综合性的 **Python 单脚本项目集合**，涵盖人工智能、机器学习、数据分析、仿真模拟、智能系统等多个领域。每个脚本独立完整，即开即用。

## 🚀 Quick Start

### 环境要求

```bash
Python 3.7+
pip package manager
```

### 安装依赖

```bash
# 核心依赖
pip install numpy scipy pandas matplotlib

# 机器学习
pip install scikit-learn tensorflow torch

# GUI框架
pip install PyQt5 PyQtChart

# 其他工具
pip install requests beautifulsoup4 joblib
```

### 运行脚本

```bash
# 直接运行任意脚本
python script_name.py

# 示例
python 神经元.py
python 智能咖啡.py
python 宇宙变迁.py
```

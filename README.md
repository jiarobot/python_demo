## 📖 Overview

一个涵盖多领域的 **Python 单脚本集合**，包含人工智能、机器学习、数据分析、仿真模拟、智能系统等多个领域的实用代码示例。每个脚本都是独立完整的，可以直接运行使用。

这些脚本涵盖了从基础算法到复杂系统的各种实现，适合学习、参考和快速原型开发。

## ✨ Highlights

- 🧠 **人工智能与机器学习** - 神经网络、遗传算法、决策树等
- 🏠 **智能系统** - 智能家居、智能咖啡、智能餐馆等
- 🔬 **科学计算** - 微分方程、量子计算、数据分析
- 🎮 **仿真模拟** - 人生仿真、宇宙变迁、生态系统
- 📊 **数据可视化** - 图表、仪表板、实时监控
- 🔧 **工具实用** - 建图系统、任务生成、网络管理

## 🚀 Quick Start

### 环境要求

```bash
Python 3.7+
pip package manager
```

### 安装依赖

大部分脚本依赖常用的Python库：

```bash
# 核心科学计算
pip install numpy scipy pandas matplotlib seaborn

# 机器学习
pip install scikit-learn tensorflow torch

# GUI框架
pip install PyQt5 PyQtChart

# 其他工具
pip install requests beautifulsoup4
```

### 运行脚本

```bash
# 直接运行任意脚本
python 神经元.py
python 智能咖啡.py
python 宇宙变迁.py

# 部分脚本支持命令行参数
python 智能咖啡.py --help
```
